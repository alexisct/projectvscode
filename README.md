# projectvscode

This repository contains a focused literature-review workspace for Nicolas Brantut's research, especially his work on:

- fault stability
- induced seismicity
- modelling approaches
- the feedback loop between laboratory experiments and models

## What's here

- `AGENTS.md`: project rules, scope, and review standards
- `brantut_research_plan.md`: research workflow and checklist
- `brantut_corpus_notes.md`: publication corpus and first-pass triage
- `brantut_reading_notes.md`: structured notes from the paper-reading phase
- `chat_transcript.md`: markdown copy of the user-assistant chat used to set up the project
- `DESCRIPTION.md`: short repository description

## Purpose

The goal of this project is to build an executive-level synthesis of Brantut's work that is grounded in direct paper reading rather than metadata alone. The review is designed to prioritize mechanism-level understanding over a generic bibliography dump.

## Current focus

The current reading and synthesis effort prioritizes papers on:

- fluid injection and fault reactivation
- pore-pressure evolution on and off faults
- permeability evolution during fault growth
- rupture stability, slip pulses, and fracture energy
- the use of fracture mechanics and elastodynamic models to interpret laboratory observations
