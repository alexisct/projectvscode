## Paper Notes

### `10.1029/2018JB015636`

- Focus: porous limestone deformation
- Microstructural relevance: permeability, porosity, and wave speed decouple because different parts of the pore and crack network control them
- Main value for synthesis: shows that physical-property evolution cannot be understood without distinguishing microcracks from flow-carrying pores

### `10.1029/2018JB016255`

- Focus: brittle deformation in antigorite serpentinite
- Microstructural relevance: highly localized mode II shear microcracking along cleavage, with little stress-induced anisotropy before failure
- Main value for synthesis: shows that mineral-specific crack geometry can change both deformation style and seismic signature

### `10.1029/2018JB016494`

- Focus: high-pressure compaction in granular quartz aggregates
- Microstructural relevance: grain chipping and crushing without strong localization
- Main value for synthesis: links compaction style to the availability of grain-scale reorganization degrees of freedom

### `10.1029/2019JB018970`

- Focus: cyclic antigorite deformation
- Microstructural relevance: sliding cracks as the source of hysteresis and inelastic dissipation
- Main value for synthesis: strong example of a constitutive model explicitly grounded in crack-scale mechanics

### `10.1029/2020JB019860`

- Focus: off-fault damage during quasi-static and dynamic rupture
- Microstructural relevance: localized microfracture damage quantified alongside seismic tomography
- Main value for synthesis: ties changing microstructure directly to rupture energetics

### `10.1029/2020JB021301`

- Focus: wave-speed recovery in Carrara marble
- Microstructural relevance: time-dependent mechanical crack closure and increased contact area within crack arrays
- Main value for synthesis: shows that "damage recovery" can be a mechanically active process at the crack scale

### `10.1016/j.epsl.2020.116179`

- Focus: pore-pressure drop during dynamic rupture
- Microstructural relevance: rupture-induced dilatancy interpreted as crack-space creation and fluid-volume change
- Main value for synthesis: direct bridge between rupture microphysics and transient fault restrengthening

### `10.1016/j.epsl.2021.117174`

- Focus: dilatancy-stabilized shear failure
- Microstructural relevance: stabilization tied to dilation, fluid storage, and possible pressure collapse or vaporization
- Main value for synthesis: shows how local internal structure enters directly into rupture-mode selection

### `10.1029/2021JB022239`

- Focus: slow rupture modelling
- Microstructural relevance: dilation enters the model as a microphysical mechanism for tip strengthening
- Main value for synthesis: connects crack-scale fluid effects to rupture-scale toughness

### `10.1029/2023JB026458`

- Focus: grain-size effects during semi-brittle flow of calcite rocks
- Microstructural relevance: grain size influences the balance between brittle cracking and crystal-plastic accommodation
- Main value for synthesis: helps link microstructural scale to rheological behavior in the semi-brittle regime

### `10.1029/2024JB028829`

- Focus: opposite pore-pressure changes on and off fault
- Microstructural relevance: partition between on-fault dilatancy and off-fault compactive crack closure
- Main value for synthesis: shows that different microstructural modes can create opposite hydraulic signals simultaneously

### `10.1029/2024JB029057`

- Focus: permeability evolution during fault growth
- Microstructural relevance: partial fracture connectivity in the cohesive zone creates strong local transmissivity heterogeneity
- Main value for synthesis: directly connects damage architecture to fluid-flow structure

### `10.1029/2024JB030022`

- Focus: permeability heterogeneity during sandstone compaction
- Microstructural relevance: connectivity of compacted zones controls whether permeability reduction becomes system-spanning
- Main value for synthesis: emphasizes that structure, not just amount of compaction, controls hydraulic outcome

### `10.1029/2024JB030914`

- Focus: microstructural evolution of Carrara marble
- Microstructural relevance: twins, lattice curvature, dislocations, and intragranular fractures evolve systematically with strain and temperature
- Main value for synthesis: strong basis for microphysically motivated constitutive modelling

### `10.1098/rspa.2024.0189`

- Focus: semi-brittle hardening model
- Microstructural relevance: microcracks treated as dislocation sinks; dislocation storage drives hardening
- Main value for synthesis: unusually explicit attempt to turn microstructure into constitutive state variables

### `10.1038/s41467-024-55363-y`

- Focus: shear localization and rupture dynamics
- Microstructural relevance: inner-fault localization controls macroscopic rupture behavior
- Main value for synthesis: clearest multiscale expression of Brantut's microstructure-to-rupture philosophy
